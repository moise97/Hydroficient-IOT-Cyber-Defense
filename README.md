
# Hydroficient IoT Cyber Defense

> **8-Week Cybersecurity Externship** | Securing smart water infrastructure through hands-on threat modeling, pipeline hardening, and AI-driven anomaly detection.

---

## Overview

In 2016, the Mirai botnet exploited weak default passwords on over 100,000 IoT devices — taking down Twitter, Netflix, Reddit, and Airbnb in a single attack. Nearly a decade later, the same vulnerabilities still exist, except now they control physical infrastructure.

This externship places me inside **Hydroficient's smart water management platform**, where sensors stream data every 0.1 seconds and a single breach could disrupt critical operations. The mission: think like an attacker first, then become the defender — intercepting data streams, spoofing commands, and identifying real security weaknesses before encrypting pipelines, locking down device access, and building AI-driven detection to flag suspicious behavior.

## What I'm Building

### 1. Threat Modeling
Map Hydroficient's IoT architecture, identify data flows, and perform a STRIDE analysis to surface the top security risks across the platform.

### 2. Secure IoT Pipeline
Build a mock IoT sensor and MQTT data pipeline, then harden it with TLS encryption and device certificate authentication.

### 3. Benchmark & Test
Run performance experiments comparing secure vs. insecure pipelines and test system behavior against expired or forged certificates.

### 4. Attack Simulation
Execute replay attack simulations and tune defenses using time-based and counter-based detection windows with HMAC signatures.

### 5. Security Dashboard & Report
Develop a real-time security monitoring dashboard in Streamlit and compile all findings into a **Security Improvement Report** for Hydroficient.

## About Hydroficient

Hydroficient is a climate technology company building automated systems to optimize water usage in commercial and industrial buildings across North America. Their IoT-enabled **HydroLogic platform** dynamically adjusts water delivery based on actual demand, typically reducing water consumption and sewage output by ~20%. Awarded the **Canadian Federal Government's ElevateIP Grant**, Hydroficient works with major restaurant chains, hotels, gyms, schools, and large facilities used by millions of people daily.
